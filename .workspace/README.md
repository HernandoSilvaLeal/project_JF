# 📂 .workspace - Archivos Temporales

Esta carpeta contiene archivos temporales y de tracking del proyecto.

---

## 📋 Contenido

```
.workspace/
├── tracked_files.txt       # Lista de archivos trackeados
├── ignored_now.txt         # Lista de archivos ignorados temporalmente
└── temp/                   # Archivos temporales varios
```

---

## 🚫 .gitignore

**IMPORTANTE:** Esta carpeta está en `.gitignore` - su contenido NO se sube a Git.

---

## 🎯 Propósito

Almacenar archivos de utilidad temporal:
- Listas de tracking de archivos
- Notas de desarrollo
- Archivos de trabajo temporales
- Cache y logs locales

---
