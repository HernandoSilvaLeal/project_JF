# 📂 CORES - Subsistemas Arquitectónicos

Esta carpeta contiene los **9 Cores** que estructuran toda la operación de RFJ 2026.

---

## 🏗️ Arquitectura de Cores

```
cores/
├── 01_doctrinal/           # Core 1: Fundamento teológico
├── 02_operativo/           # Core 2: Escalera de servicios S0-S8
├── 03_misional/            # Core 3: Taxonomía de frentes F1-F5
├── 04_producto/            # Core 4: 99 Servilletas como MVP
├── 05_alcance/             # Core 5: Círculos doctrinales C0-C3
├── 06_canales/             # Core 6: Distribución multicanal
├── 07_plataforma/          # Core 7: SaaS web platform
├── 08_agentes_ia/          # Core 8: Orquestación de IA
└── 09_ingenieria/          # Core 9: CI/CD y DevOps
```

## 📋 Descripción de Cada Core

### 01_doctrinal/ - Kernel Bíblico
**Propósito:** Fundamento teológico convergente  
**Estado:** ✅ En construcción (S001 completa, S002-S099 pendientes)  
**Contenido:**
- Kernel Doctrinal (99 servilletas)
- Convergencia entre tradiciones sanas
- Trazabilidad bíblica total

### 02_operativo/ - Escalera de Servicios
**Propósito:** Estructura operacional S0-S8  
**Estado:** 📋 Documentado  
**Contenido:**
- S0: Evangelismo básico (TikTok, YouTube)
- S1: Discipulado inicial
- S2-S7: Servicios progresivos
- S8: Plantación de iglesias

### 03_misional/ - Taxonomía de Frentes
**Propósito:** Mapeo exhaustivo de necesidades  
**Estado:** 📋 Taxonomía completa  
**Contenido:**
- F1: Individuo (A1-A4: Identidad/Propósito/Sufrimiento/Muerte)
- F2: Familia (A5-A8: Matrimonio/Crianza/Finanzas/Crisis)
- F3: Comunidad (A9-A12: Iglesia/Trabajo/Ciudadanía/Cultura)
- F4: Conocimiento (A13-A16: Epistemología/Ética/Apologética/Teología)
- F5: Trascendencia (A17-A20: Santificación/Misión/Perseverancia/Gloria)

### 04_producto/ - 99 Servilletas
**Propósito:** Definición del producto MVP  
**Estado:** 🚧 En diseño  
**Contenido:**
- Roadmap S001-S099
- Priorización por convergencia
- Estrategia de lanzamiento

### 05_alcance/ - Círculos Doctrinales
**Propósito:** Niveles de consenso teológico  
**Estado:** 📋 Definido  
**Contenido:**
- C0: Evangelio (100% convergencia)
- C1: Esenciales (80-100% convergencia)
- C2: Importantes (50-79% convergencia)
- C3: Periféricos (<50% convergencia)

### 06_canales/ - Distribución Multicanal
**Propósito:** Estrategia de distribución  
**Estado:** 🚧 En análisis  
**Contenido:**
- TikTok (videos cortos)
- YouTube (videos largos)
- Instagram (posts/reels)
- Podcast
- Blog/Web

### 07_plataforma/ - SaaS Platform
**Propósito:** Plataforma web centralizada  
**Estado:** 🔜 Planificado  
**Contenido:**
- Dashboard de contenido
- Gestión de servilletas
- Analytics y métricas
- API REST

### 08_agentes_ia/ - Orquestación IA
**Propósito:** Automatización inteligente  
**Estado:** 🔜 Planificado  
**Contenido:**
- Generación de guiones
- Edición de contenido
- Distribución automatizada
- Análisis de engagement

### 09_ingenieria/ - DevOps & CI/CD
**Propósito:** Infraestructura técnica  
**Estado:** 🔜 Planificado  
**Contenido:**
- Pipeline CI/CD
- Testing automatizado
- Deployment
- Monitoring
