## 1️⃣ TESIS CENTRAL (1 frase)

**Regla:** Debe poder decirse en 10-15 segundos. Clara y bíblica.

**Tesis:**
```
La Biblia enseña que [AFIRMACIÓN CENTRAL].

Ejemplo: "La Biblia enseña que Dios es absolutamente santo y no puede tolerar el pecado."
```

## 4️⃣ VERIFICACIÓN GUIADA (Pasos para comprobar con Biblia)

**Objetivo:** Que el usuario verifique por sí mismo la verdad de esta servilleta.

**Pasos:**
```
PASO 1: Lee [Texto ancla 1]
        Observa: [Qué debe notar el usuario aquí]

PASO 2: Lee [Texto ancla 2]
        Observa: [Qué debe notar el usuario aquí]

PASO 3: Conecta ambos textos
        Pregunta: [¿Qué te dicen estos textos juntos sobre la tesis?]

PASO 4: Confirma la tesis
        [Verifica si la tesis central se sostiene con estos textos]

PASO 5 (opcional): Lee contexto ampliado
        [Sugerencia de capítulo completo o pasaje relacionado]
```

## 5️⃣ APLICACIÓN (Evangelio → Vida)

**Regla:** No moralismo. La aplicación debe colgar del evangelio, no reemplazarlo.

**Estructura:**
- ¿Qué revela esto sobre Dios?
- ¿Qué revela sobre el hombre?
- ¿Cómo apunta a Cristo?
- ¿Qué cambia en mi vida si esto es verdad?

**Aplicación (3-5 bullets):**
```
- [Aplicación 1: Cómo afecta tu comprensión de Dios]

- [Aplicación 2: Cómo afecta tu comprensión de ti mismo]

- [Aplicación 3: Cómo afecta tu confianza en Cristo]

- [Aplicación 4 (opcional): Cómo afecta tu conducta diaria]
```

## ✅ CHECKLIST DE CALIDAD

Antes de marcar como PUBLICADO, verifica:

### Contenido:
- [ ] Tesis clara en 1 frase (10-15 segundos de lectura)
- [ ] 2-6 textos ancla con referencias exactas (Libro Cap:Vers)
- [ ] Lógica de conexión explícita (por qué estos textos sostienen la tesis)
- [ ] Verificación guiada tiene 3-5 pasos claros
- [ ] Aplicación conecta con evangelio (no moralismo)
- [ ] 2-4 errores comunes identificados

### Doctrina:
- [ ] Centro (Dios/Hombre/Cristo) presente
- [ ] No deriva a periferia como centro
- [ ] Lenguaje para no iniciados (jerga explicada)
- [ ] Textos en contexto (no descontextualizados)

### Formato:
- [ ] ID asignado (S0XX)
- [ ] Metadata completa
- [ ] Tags y clasificación
- [ ] Puentes a otras servilletas
- [ ] Diagrama descrito (aunque no esté dibujado aún)

### Guardarraíles:
- [ ] Si riesgo ALTO: disclaimers y derivación responsable
- [ ] Si toca controversia: aclarar que es C2 o C3 (no centro)
- [ ] Si toca sufrimiento: incluir esperanza bíblica

## 🎯 CRITERIO DE ÉXITO PEDAGÓGICO

**Una servilleta funciona si:**
1. Una persona puede explicar la tesis en 30-60 segundos
2. Puede señalar 2 textos ancla de memoria
3. Entiende la diferencia entre esta verdad y un "cliché cristiano"
4. Sabe dónde ir después (siguiente servilleta o frente)
