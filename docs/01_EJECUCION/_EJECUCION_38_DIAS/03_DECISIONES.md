## 🔒 PRINCIPIOS DE DECISIÓN

1. **Decidir con data cuando haya; con prudencia cuando no:**
   - Si métricas claras → ajustar
   - Si incertidumbre → ir con lo más simple y bíblicamente sólido

2. **No re-decidir sin razón:**
   - Una vez tomada, ejecutar mínimo 2-4 semanas antes de cambiar
   - Excepción: si hay falla evidente inmediata

3. **S0 manda siempre:**
   - Si una decisión apaga el fuego espiritual → revisar
   - Si una decisión te aleja de oración → mala decisión

4. **Shipped > Perfect:**
   - Preferir decisión que permite acción sobre decisión "perfecta" que paraliza

5. **Documentar aprendizajes:**
   - Cada decisión debe tener "¿qué aprendí?" al final del sprint
